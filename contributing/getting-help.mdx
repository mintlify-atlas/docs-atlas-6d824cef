If you have questions about the Worlds Platform, need to report a bug, or want to request a feature, the channels below are the right places to start.

## Before you ask

To get the fastest resolution:

1. **Search the documentation**: Use the search bar to look for your topic across all docs.
2. **Search existing issues**: Browse [GitHub Issues](https://github.com/wazootech/worlds/issues) for similar problems or feature discussions.
3. **Check the guides**: The [guides section](/guides/querying) covers common usage patterns including SPARQL queries, RAG workflows, and agent integration.

## Reporting bugs

When opening a bug report on GitHub, include:

- A clear, descriptive title.
- The version of Deno and any relevant package versions.
- Steps to reproduce the issue with a minimal example.
- The actual behavior and the expected behavior.
- Any relevant error messages or logs.

<Warning>
  Do not include API keys, credentials, or other secrets in bug reports or issue comments.
</Warning>

## Requesting features

Feature requests are welcome. Before submitting one:

- Check existing issues to see if the feature has already been requested.
- Describe the use case that motivates the request, not just the implementation.
- Explain how the feature fits within the existing architecture.
